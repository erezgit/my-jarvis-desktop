# Changelog

## [1.0.0] - 2025-09-08

### Added
- Initial release of My Jarvis Desktop
- Three-panel layout with file explorer, document preview, and terminal
- Fully functional terminal using xterm.js and node-pty
- WebGL acceleration for terminal rendering
- Support for bash shell
- Fixed duplicate character issue by using bash instead of zsh
- Cross-platform support (macOS, Windows, Linux)

### Technical Stack
- Electron v37.4.0
- React v19.1.1
- TypeScript v5.9.2
- xterm.js v5.5.0
- node-pty v1.0.0
