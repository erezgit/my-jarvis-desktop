# Welcome to My Jarvis Desktop

## MDX Preview System

This is a **live MDX preview** with hot reloading capabilities. You can edit this file and see changes instantly!

### Features

- 🚀 **Fast Refresh**: Changes appear instantly
- 📝 **Rich Markdown**: Full GitHub Flavored Markdown support
- ⚛️ **React Components**: Embed interactive components
- 🎨 **Syntax Highlighting**: Beautiful code blocks
- 🔥 **Hot Module Replacement**: No app restart needed

### Code Example

```javascript
function greet(name) {
  return `Hello, ${name}! Welcome to My Jarvis Desktop.`
}

console.log(greet('User'))
```
